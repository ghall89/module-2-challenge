About Me

I spent my childhood tinkering with computers, and eventually taught myself how to create very basic websites using the limited tools available in the late 90's and early 2000's. My first real, published website was, in typical nerd fashion, a Star Trek fan site. Though I found myself in the realm of graphic design, I still had an interest in web development. A string of circumstances and opportunities led me to attend a coding bootcamp where I'm learning to take my passions to the next level.

When I'm not writing code I can be found going on hikes, sketching, or playing video games. My weaknesses include writing about myself, so this is a very rough draft of my "About Me" section.
